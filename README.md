# Financial Dashboard

A simple yet powerful **web-based dashboard** built with **Python and Streamlit** for managing and visualizing personal finance. Track your **income**, **expenses**, and **savings** with interactive charts and CSV-based data.

---

## Features

- Web interface powered by **Streamlit**.
- Import and manage financial data from CSV files.
- Visualize income, expenses, and savings trends with interactive charts.
- Clean and modular Python code following best practices.
- Environment variables support using `.env`.
- Beginner-friendly and extendable for future enhancements.

---

## Requirements

- Python 3.10+
- Streamlit
- Pandas
- Matplotlib / Seaborn
- python-dotenv (for `.env` support)
